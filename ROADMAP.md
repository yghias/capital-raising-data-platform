# Roadmap

## Near term
- add more robust schema-contract fixtures for each source connector
- package the entity resolution service behind a FastAPI endpoint
- add warehouse-specific deployment examples for Snowflake and Databricks
- expand semantic metric definitions and KPI reconciliation checks

## Mid term
- productionize Neo4j graph loads and relationship path APIs
- incorporate LLM-assisted signal classification with human review
- support event-driven signal ingestion for near-real-time alerts
- add feature store patterns for investor scoring experiments

## Long term
- embed prescriptive next-best-action recommendations into CRM
- integrate meeting transcripts and notes with appropriate privacy controls
- expose governed data products through internal APIs and reverse ETL
